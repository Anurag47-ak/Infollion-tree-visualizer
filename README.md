# Infollion-tree-visualizer

React Flow tree renderer with centered parent-child layout, clean spacing, and expand/collapse interaction.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production Build

```bash
npm run build
```

## Notes

- The sample tree is defined in `src/App.tsx`.
- The layout recalculates automatically when nodes collapse or expand.
- Nodes can be selected by clicking the card body and collapsed with the badge button.
